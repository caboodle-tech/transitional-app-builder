Demo Page

Hello, this is a simple demo page to get you started. If you look at the source file for this page app/index.html you will see the following variables:

  • t:head – A template file called head. This file is located in private/templates and contains all the HTML code for every pages <head> section.
  • f:copyright(2021) – A function called copyright. This file is located in private/functions. You can provided it the year your app/ company was founded and it will create the correct copyright footer for you.

If you look inside the head template file you will notice more advanced variables:

  • f:title(g:title,v:title) – A function called title that accepts two parameters. You can provided a global title (g:title) and a local title (v:title) for the function to use. This file is located in private/functions.
  • f:mainStylesheet –A function called mainStylesheet that automatically adds the <link> to our main stylesheet. This file is located in private/functions.
